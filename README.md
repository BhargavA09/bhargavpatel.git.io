## 🖥️ About This Portfolio

This is my personal portfolio website — built from scratch with vanilla HTML, CSS, and JavaScript. It covers my background, skills, work experience, and contact details, and is hosted for free via GitHub Pages.

The site features animated skill bars, a typed-text hero section, scroll-triggered AOS animations, a live contact form, and a mobile-responsive layout — all without any JavaScript framework.

---

## ✨ Features

| Feature | Details |
|---|---|
| 🎯 Typed hero animation | Rotating role titles using `typed.js` |
| 📊 Animated skill bars | Visual proficiency indicators with smooth fill |
| 🔢 Live counters | Animated stats (clients, projects, hours coded) |
| 📱 Fully responsive | Mobile-first layout using Bootstrap 5 |
| ✉️ Contact form | PHP-powered email form with validation |
| 🗺️ Google Maps embed | Live location map in the contact section |

## 🚀 Run Locally

No build step required — this is plain HTML/CSS/JS.

```bash
# Clone the repo
git clone https://github.com/BhargavA09/bhargavpatel.git.io.git
cd bhargavpatel.git.io

# Open in browser directly
open index.html

# Or serve with a local server (recommended for contact form)
npx serve .
```

> **Note:** The contact form (`forms/contact.php`) requires a PHP server to send emails. For local testing without PHP, you can use [XAMPP](https://www.apachefriends.org/) or [Laragon](https://laragon.org/).

## 🌐 Deploy to GitHub Pages

This site is already configured for GitHub Pages. To deploy your own fork:

1. Fork this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Your site will be live at `https://<your-username>.github.io/<repo-name>`
